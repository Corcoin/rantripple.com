DHCP Server Configuration


  1. Configure your DHCP server with the following details:
  2. Server IP Address: 192.168.1.1
  3. Client IP Range: 192.168.1.100 - 192.168.1.200
  4. Subnet Mask: 255.255.255.0
  5. Default Gateway: 192.168.1.1
  6. DNS Server: 8.8.8.8 (Google's public DNS server)

Client Configuration Instructions


Windows

  1. Go to Settings > Network & Internet > Proxy.
  2. Enable "Use a proxy server".
  3. Enter the Address: 192.168.1.78 and Port: 8088.
  4. Click "Save" to apply the settings.

Mac

  1. Open System Preferences > Network.
  2. Select your network connection and click "Advanced".
  3. Go to the Proxies tab.
  4. Check "Web Proxy (HTTP)" and "Secure Web Proxy (HTTPS)".
  5. Enter the Address: 192.168.1.78 and Port: 8088.
  6. Click "OK" and "Apply".

Raspberry Pi

  1. Edit the /etc/environment file: sudo nano /etc/environment.
  2. Add the following lines: 
    http_proxy="http://192.168.1.78:8088"
https_proxy="http://192.168.1.78:8088"
  3. Save the file and reboot your Raspberry Pi.

Linux

  1. Open Terminal and edit the environment file: sudo nano /etc/environment.
  2. Add the following lines: 
    http_proxy="http://192.168.1.78:8088"
https_proxy="http://192.168.1.78:8088"
  3. Save the file and run the command source /etc/environment to apply the settings.
